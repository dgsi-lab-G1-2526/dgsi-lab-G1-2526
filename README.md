# dgsi-lab-G1-2526
Proyecto GitHub para desarrollar los Sprints del laboratorio de la asginatura Desarrollo y Gestión de Sistemas de Información

Sprint Review – Sprint 1 (P1)
Durante el Sprint 1 se definió el alcance inicial del proyecto y se establecieron las bases conceptuales del modelo de negocio y del proceso E2E seleccionado. 
Se identificó el problema a resolver en el ámbito de la ciberseguridad en redes y se formularon las primeras historias de usuario y entregables respecto a la documentación. 
Además, se estructuró el repositorio, se organizó el tablero Agile y se definieron los criterios de terminado para cada ítem. (Dentro de las propias issues).
El objetivo del Sprint 1 fue construir el marco conceptual del proyecto y dejar preparado el entorno de trabajo para su evolución estratégica en el Sprint 2. 
Todos los entregables definidos en el sprint fueron completados conforme a los criterios establecidos.

Sprint Review – Sprint 2 (P2)
Durante el Sprint 2 se ha trabajado el enfoque estratégico del proyecto, identificando las principales presiones competitivas del entorno y definiendo 
respuestas estratégicas alineadas con la propuesta de valor en ciberseguridad. Se ha analizado el proceso actual (As-Is) y diseñado un proceso objetivo (To-Be),
incorporando mejoras en resiliencia, disponibilidad y gobernanza del dato. Además, se han definido KPIs con baseline, riesgos estratégicos y una métrica de 
calidad del dato, junto con un blueprint conceptual del Sistema de Información de Apoyo (SIA).
